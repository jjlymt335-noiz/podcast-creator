2. 这是一个 Podcast Creator（广播剧创作工具）项目，技术栈：
   - React 18 + TypeScript + Vite
   - Tailwind CSS + shadcn/ui
   - Zustand 状态管理
   - Axios API 调用

3. 项目的关键功能：
   - 文档上传与解析（PDF/EPUB/DOCX/TXT/HTML）
   - Auto-assign voices（AI 自动识别说话人并匹配音色）
   - 手动音色选择（Host/Guest）
   - Voice Library 集成
   - Voice Design 功能（文本描述生成音色）
   - 文本编辑和 TTS 合成

4. 最重要的文件：
   - src/components/EntryPage/index.tsx - 入口页面（双模式：Auto-assign / Manual）
   - src/components/EntryPage/VoiceSelector.tsx - Host/Guest 音色选择器
   - src/components/EditorPage/VoicePanel.tsx - 说话人音色面板
   - src/components/EditorPage/TextEditor.tsx - 文本编辑器（支持 Voice Design）
   - src/components/VoiceLibrary/VoiceLibraryModal.tsx - 音色库弹窗
   - src/components/VoiceLibrary/VoiceDesignModal.tsx - Voice Design 弹窗
   - src/store/project-store.ts - 项目状态管理
   - src/lib/api/ - 所有 API 封装
